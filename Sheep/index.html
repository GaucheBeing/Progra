Determinar el dominio de \(f(x)=\frac{x}{\ln(x^2)}\)

\((-\infty, -1)\) \((-1, 0)\) \((0, 1)\) \((1, \infty)\)
\(x\) - - + +
\(\ln(x^2)\) + - - +
\(\frac{x}{\ln(x^2)}\) - + - +

\(\therefore\) podemos decir que \(f(x) < 0 \iff x \in (-\infty,-1) \cup (0,1)\) y que \(f(x) > 0 \iff x \in (-1,0) \cup (1,\infty)\)
